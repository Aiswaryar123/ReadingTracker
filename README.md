# Bookshelf & Reading Tracker

A full-stack application to manage your library, track reading progress, and set yearly goals.

## Tech Stack

- **Backend:** Go (Gin Framework)
- **Database:** PostgreSQL (GORM)
- **Frontend:** React
- **Architecture:** Handler-Service-Repository pattern

## Features

- [x] Manual Book Cataloging
- [x] Reading Progress Tracking (Pages/Status)
- [x] Book Reviews & Ratings
- [x] Yearly Reading Goals & Progress
- [ ] ISBN Lookup (Coming soon)

## How to Run

1. Setup PostgreSQL database.
2. Configure `.env` file.
3. `cd backend && go run main.go`
4. `cd frontend && npm start`
